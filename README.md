# Readme

####Register user

- POST /api/users
- Body example:
```
{
	"user": {
		"email": "olaaaaa@oale.com",
		"password": "asdasasad",
		"username": "olla"
	}
}
```

####Login user

- POST /api/users/sign_in
- Body example:
```
{
	"user": {
		"email": "olaaaaa@asfasoale.com",
		"password": "asdasasad"
	}
}
```
